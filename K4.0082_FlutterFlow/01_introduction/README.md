# Module 01: Introduction & Setup

## Overview

This introductory module establishes the foundational understanding of FlutterFlow and no-code/low-code development. It covers the core concepts of app architecture, the FlutterFlow platform setup, and familiarization with the development environment.

**Key Topics:**
- Understanding no-code and low-code development paradigms
- App architecture fundamentals (Frontend, Backend, APIs)
- FlutterFlow platform account setup and configuration
- Dashboard navigation and workspace organization
